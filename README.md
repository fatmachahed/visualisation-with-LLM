## Visualisation with LLM

## Description
**visualisation-with-llm** is a library for automatic data visualization generation using Large Language Models (LLMs) like Google Gemini. It helps users upload datasets, define problems, and automatically generate, customize, and export visualizations.

## Features
- **Automatic Visualization Generation**: Leverages LLMs to propose relevant visualizations based on dataset and user problem statement.
- **Customization Options**: Users can adjust visualization titles, axes, and chart types to meet their needs.
- **Interactive Data Filtering**: Allows users to filter data by column values before generating visualizations.
- **Wide Range of Visualizations**: Supports scatter plots, line charts, bar charts, histograms, boxplots, heatmaps, and more.
- **Data Summarization**: Automatically provides summary statistics of the dataset, including numeric and categorical details.
- **Export Options**: Users can download visualizations as PNG, datasets as CSV, and configurations as JSON files.
- **Streamlit Web Interface**: Built with Streamlit for an interactive and user-friendly web-based application.
